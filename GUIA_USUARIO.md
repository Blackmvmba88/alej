# Programa Educativo para Estudiantes de Secundaria y Preparatoria

## 📚 Descripción

Este es un programa educativo interactivo diseñado específicamente para estudiantes de nivel secundaria y preparatoria. El programa ofrece lecciones, ejercicios y quizzes en cuatro materias principales:

- **Matemáticas** (Álgebra, Geometría, Trigonometría)
- **Ciencias** (Física, Química, Biología)
- **Literatura y Comprensión Lectora**
- **Historia** (Universal y de América)

## 🎯 Características

### Sistema de Puntos
- Gana puntos al completar lecciones y ejercicios correctamente
- Sigue tu progreso en cada materia
- Alcanza diferentes niveles: Principiante, Intermedio, Avanzado, Experto

### Persistencia de Datos
- Tu progreso se guarda automáticamente
- Puedes cerrar y reabrir el programa sin perder tu avance
- Cada estudiante tiene su propio perfil

### Contenido Educativo
- Lecciones teóricas claras y concisas
- Ejercicios prácticos con retroalimentación inmediata
- Quizzes para evaluar tu conocimiento
- Explicaciones detalladas cuando respondes incorrectamente

## 🚀 Cómo Usar el Programa

### Requisitos
- Python 3.6 o superior
- No requiere bibliotecas externas (solo usa módulos estándar de Python)

### Instalación
1. Asegúrate de tener Python instalado en tu computadora
2. Descarga el archivo `programa_educativo.py`

### Ejecución
En la terminal o línea de comandos, ejecuta:

```bash
python3 programa_educativo.py
```

O en Windows:
```bash
python programa_educativo.py
```

### Primer Uso
1. Al iniciar el programa, te pedirá tu nombre
2. Se creará automáticamente un perfil para ti
3. Tu progreso se guardará en el archivo `progreso_estudiante.json`

## 📖 Estructura del Programa

### Menú Principal
Desde el menú principal puedes acceder a:
1. **Matemáticas** - Lecciones de álgebra, geometría y trigonometría
2. **Ciencias** - Física, química y biología
3. **Literatura** - Géneros literarios, figuras retóricas y comprensión lectora
4. **Historia** - Historia universal y de América
5. **Ver mi progreso** - Consulta tus puntos y lecciones completadas
6. **Salir** - Cierra el programa (tu progreso se guarda automáticamente)

### Matemáticas

#### Álgebra - Ecuaciones Lineales
- Aprende a resolver ecuaciones lineales paso a paso
- Ejercicios prácticos con explicaciones detalladas
- Ejemplo: `2x + 5 = 13`

#### Geometría - Áreas y Volúmenes
- Fórmulas de áreas: cuadrado, rectángulo, triángulo, círculo
- Fórmulas de volúmenes: cubo, cilindro, esfera
- Problemas prácticos de aplicación

#### Trigonometría - Funciones Básicas
- Definiciones de seno, coseno y tangente
- Valores importantes de funciones trigonométricas
- Identidades fundamentales

#### Ejercicios Prácticos
- Serie de problemas para practicar lo aprendido
- Retroalimentación inmediata
- Puntos por cada respuesta correcta

### Ciencias

#### Física - Leyes de Newton
- Las tres leyes fundamentales del movimiento
- Fórmula F = ma y aplicaciones prácticas
- Ejemplos de la vida cotidiana

#### Química - Tabla Periódica
- Grupos importantes de elementos
- Símbolos químicos comunes
- Estructura atómica básica

#### Biología - Célula y Sistemas
- Diferencias entre células procariotas y eucariotas
- Organelos celulares y sus funciones
- Características de células vegetales y animales

#### Quiz de Ciencias
- Preguntas variadas de física, química y biología
- Evalúa tu conocimiento general en ciencias
- Gana puntos adicionales

### Literatura y Comprensión Lectora

## 📊 Sistema de Progreso

### Niveles
- **Principiante**: 0-49 puntos
- **Intermedio**: 50-149 puntos
- **Avanzado**: 150-299 puntos
- **Experto**: 300+ puntos

### Puntuación
- Lecciones completadas: 10 puntos
- Ejercicios prácticos: 5 puntos por pregunta correcta
- Quizzes: 5 puntos por pregunta correcta

### Archivo de Progreso
El programa guarda tu progreso en `progreso_estudiante.json` con:
- Fecha de registro
- Puntos por materia
- Lecciones completadas
- Historial de avance

## 💡 Consejos para Estudiantes

1. **Practica regularmente**: Dedica al menos 20-30 minutos diarios al programa
2. **Revisa tus errores**: Cuando respondas incorrectamente, lee la explicación
3. **Completa todas las materias**: Un conocimiento balanceado es importante
4. **Establece metas**: Intenta alcanzar el siguiente nivel
5. **No te apresures**: Tómate tu tiempo para entender cada concepto
