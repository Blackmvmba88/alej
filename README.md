# ALEJ - Plataforma Educativa Gamificada

🎓 **A**prende **L**a **E**ducación **J**ugando

## Descripción

ALEJ es una plataforma educativa innovadora diseñada para impulsar la creatividad y el pensamiento crítico en jóvenes a través de módulos interactivos de ciencias, arte y tecnología. La plataforma utiliza gamificación para hacer el aprendizaje más atractivo y efectivo.

## Características Principales

### 🔬 Módulo de Ciencias
- **Física Interactiva**: Simulaciones de péndulos, movimiento proyectil y ondas
- **Matemáticas Dinámicas**: Retos de álgebra, geometría y cálculo
- **Experimentos Virtuales**: Densidad y flotación, óptica, conservación de energía

### 🎨 Módulo de Arte
- **Dibujo Digital**: Canvas interactivo para expresión creativa
- **Patrones y Formas**: Creación de diseños geométricos
- **Historia del Arte**: Exploración de movimientos artísticos
- **Música y Sonido**: Composición y teoría musical

### 💻 Módulo de Tecnología
- **Programación**: Scratch visual, Python básico, desarrollo web
- **Robótica Virtual**: Simulación y programación de robots
- **Pensamiento Computacional**: Puzzles lógicos y algoritmos

## Sistema de Gamificación

- 🏆 **Puntos**: Gana puntos completando actividades y retos
- ⭐ **Niveles**: Avanza de nivel al acumular puntos
- 🎖️ **Insignias**: Desbloquea logros especiales
- 📈 **Progreso**: Seguimiento del avance en cada módulo

## Instalación

1. Clona este repositorio:
```bash
git clone https://github.com/Blackmvmba88/alej.git
cd alej
```

2. Abre `index.html` en tu navegador web favorito

No se requieren dependencias adicionales - la aplicación funciona completamente en el navegador.

## Uso

1. Abre la aplicación en tu navegador
2. Explora los tres módulos principales: Ciencias, Arte y Tecnología
3. Haz clic en las actividades que te interesen
4. Completa retos y experimentos para ganar puntos
5. Desbloquea logros y sigue tu progreso en "Mi Perfil"
